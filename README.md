# WEBデザイン学習資料

個人開発に必要なWEBデザインの用語をまとめた学習用教材です。基礎から実践まで、体系的にWEBデザインを学べます。

## 📚 コンテンツ

### [用語集](./glossary/)

1. [基礎用語](./glossary/01-basics.md)
   - HTML/CSS/JavaScriptなどの基本概念
   - WEBの仕組みとブラウザ
   - 開発に必要な基礎知識

2. [デザイン原則・理論](./glossary/02-design-principles.md)
   - デザインの4大原則
   - 視覚デザインの理論
   - 情報設計の基礎

3. [レイアウト・構造](./glossary/03-layout.md)
   - グリッドシステム
   - レスポンシブデザイン
   - ページ構成要素

4. [色・タイポグラフィ](./glossary/04-color-typography.md)
   - 色彩理論
   - カラースキーム
   - フォントとタイポグラフィ

5. [UI/UX](./glossary/05-ui-ux.md)
   - ユーザーインターフェース
   - ユーザーエクスペリエンス
   - インタラクションデザイン

6. [技術・実装](./glossary/06-technical.md)
   - CSS設計手法
   - アニメーション
   - パフォーマンス最適化

## 🎯 対象者

- プログラミング初心者
- 個人開発でWEBサイトを作りたい方
- WEBデザインの基礎を学びたい方
- 体系的にデザイン用語を理解したい方

## 📖 使い方

1. まずは[基礎用語](./glossary/01-basics.md)から順番に読んでいくことをおすすめします
2. 気になる用語があれば、各カテゴリから探してください
3. 実践的な内容は[技術・実装](./glossary/06-technical.md)で学べます
