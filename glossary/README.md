## 🔍 用語の探し方

1. **順番に学習する場合**
   - 01から07まで順番に読み進めることで、体系的に学べます
   - 特に07は2024-2025年の最新トレンドなので必読です

2. **特定の用語を探す場合**
   - 各ファイル内で Ctrl+F (Command+F) で検索

3. **カテゴリから探す場合**
   - 上記のカテゴリ一覧から該当するファイルを選択

## 📊 重要度の見方

- ⭐⭐⭐ メジャー: 必ず知っておくべき基本用語
- ⭐⭐ 実用的: 実務でよく使う用語
- ⭐ ニッチ: 知っておくと便利な専門用語

## 💡 学習のヒント

- まずは⭐⭐⭐の用語を重点的に学習
- 実際にコードを書きながら用語を確認
- 不明な関連用語があれば、リンクを辿って理解を深める
